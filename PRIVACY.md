# Underdog Caddie Privacy Policy

Effective date: May 2, 2026

Underdog Caddie is a Chrome extension that helps users view fantasy draft exposure, team context, and player combo information while using Underdog Fantasy draft rooms.

## Data Collection

Underdog Caddie processes data from CSV files that users choose to import into the extension. It may also read visible player and roster information from Underdog Fantasy draft-room pages so it can display inline badges.

The extension does not collect data from unrelated websites.

## Data Storage

Imported CSV data and draft-room state are stored locally in the user's browser using Chrome extension local storage.

Underdog Caddie does not upload this data to a server.

## Data Use

Underdog Caddie uses locally stored data only to provide its core functionality, including:

- exposure percentage badges
- same-team roster badges
- historical player combo badges
- local import and draft-room status

## Data Sharing

Underdog Caddie does not sell, share, upload, or transmit user data to any external server.

The extension does not use analytics, advertising, tracking pixels, or third-party data processors.

## Account Access

Underdog Caddie does not access, modify, or control the user's Underdog Fantasy account. It does not place drafts, make picks, submit entries, or change account settings.

## Permissions

Underdog Caddie requests browser permissions only to store local extension data and interact with supported Underdog Fantasy draft-room pages for its user-facing badge features.

## Contact

For privacy questions, contact the developer through the support information listed on the Chrome Web Store listing or the project repository.
